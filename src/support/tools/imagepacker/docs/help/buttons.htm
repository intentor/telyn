Image Packer

Gadget buttons

Gadget Action
Add Add an image to the list and place in pack (if room allows)
Add (folder) Add a set of images from a folder to the list and place in pack (if room allows)
Remove Remove the selected image from the list/pack
Rename Rename selected image
Change Color Change an area of color in the image
Resize Change the size of the image
FlipX Flip the image horizontally
FlipY Flip the image vertically
Rotate Rotates the selected image 90 degrees clockwise
Setup animation Open the Animations window for setting up animation parameters on the selected image
Set image border Open the Set border window to add/remove a surrounding border for the selected image
Limit pack size Impose size limitations on the pack
