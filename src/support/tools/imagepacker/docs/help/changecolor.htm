Image Packer

 

Change color

Change the background (or mask) color of an image.

The Change color window shows 2 images.
The top image is the original and the bottom image shows a preview of any color changes made.

Click anywhere in the top image to select the color to change.
A default color of BLACK is used as a replacement color.
You can change this by clicking the color box adjacent to 'Replacement color'.

Click Apply to accept the changes or close the window via [X] to ignore.
