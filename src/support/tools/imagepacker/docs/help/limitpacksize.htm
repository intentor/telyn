Image Packer

 

Limit pack size

You may wish to set a limit for the size of the image pack.
The Limit pack size window allows you to impose limits on the pack size.

By default the pack automatically changes size to accomodate the listed sprites.
However, there is a maximum size of 2048 by 2048.

Set the width/height limits using the comboboxes.
Tick the Auto-Resize checkbox to allow Image Packer to set the pack size as required.

The sizes available are in powers of 2:

16, 32, 64, 128, 256, 512, 1024, and 2048
