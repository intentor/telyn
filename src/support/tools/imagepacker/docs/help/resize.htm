Image Packer

 

Resize image

Change the size of an image in the pack with this option.

You can change the image size in 2 ways:

a) Modify the width/height values. If the Keep aspect checkbox is ticked the width/height are changed to keep the image the same aspect.

b) Click the Reduce by buttons to scale down the image by the percentage shown.

Click Apply to accept the changes or close the window via [X] to ignore.

NOTE:
The smallest size you can set is 4 by 4 pixels.
