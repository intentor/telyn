Border Settings

When you use packed sprites there may be situations where the edges of the sprites bleed with others close together. To avoid this issue you can apply a border around your sprite.
The size of this border can be set by adjusting the Border size value (which becomes active when you select any setting other than Remove). A default value of 2 pixels is recommend.

The following border settings are available:

Border Type Action
Remove Removes the border returning the image to it's previous state
Masked Surrounds the image with a black masked border
Stretched Expands the edges of the image. For icon and tile sprites
Repeated Repeats the edges of the image.
The repeated edge is copied from the opposite side of the image

If you close the Border settings window using the [X] then any changes are ignored.
Use the Apply button to keep the changes.

 

WARNING:
This operation does not work with animated images.
