Animation settings

With this option you can set up animation frames for an image.

The are 2 ways to set up the frames in the animation:

a) Click on the bottom/right corner of the FIRST frame in the animation strip

b) Enter the amount of frames horizontally and vertically in the text boxes.

Once you have set the frames tick the Run animation checkbox.
The animation begins to play. Adjust the playback speed with the slider.

Note: Animation frames run from top/left to bottom right and loop around again.

You need a minumum value of 1 in the frame text boxes to allow animation to take place.

Clicking the [RESET] button removes all animation settings from the image.

Use the Fit image to frames button to re-scale the image to fit in with the frame settings.
This helps to create clean animations where the size of the image does not match the frame width and height values. For example, if you have an image which is 66 pixels wide with 4 animation frames across, then the ideal width should be 64 pixels which makes each of the 4 frames 16 pixels wide.
