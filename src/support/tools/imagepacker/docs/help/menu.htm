Image Packer

Menu options

File Menu Action
New Clears the current pack
Load image pack Load a previously saved image pack
Save image pack Save the current pack
File > Export images Export the individual packed images
Exit Close the program

 

Edit Menu Action
Copy image to clipboard Copies the selected image to the clipboard
Paste image from clipboard Paste the clipboard image into Image packer.
You have the option to replace the currently selected, paste as a new image, or cancel the operation

 

Prefs Menu Action
Change background color Change the background color used in the main canvas area.
Use this to examine where transparent areas of the image are
Change highlight color Changes the color of the highlight selection box
Show grid Displays an overlay grid to show animation frames and the image sizes

 

Help Menu Action
Guide Open the Image Packer guide
Keyboard shortcuts Quick access to the keyboard shortcuts file
About General information, version number, etc..
